## 📊 技术细节

### 支持的参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| **question** | string | - | 问题内容（必填） |
| **options** | string[] | - | 选项列表（有则为选择题） |
| **required** | boolean | true | 是否必须回答 |
| **initial** | number | 0 | 默认选项索引 |

---

### 返回数据

```json
{
  "success": true,
  "content": "✅ **用户回答**\n...",
  "data": {
    "question": "问题内容",
    "answer": "用户回答",
    "options": ["选项 1", "选项 2"],
    "required": true
  }
}
```

## ⚠️ 注意事项

### 1. 交互会阻塞

**说明**: prompts 会暂停程序等待用户输入

**影响**: 
- ✅ 真正的交互
- ⚠️ 会阻塞当前会话

---

### 2. 用户取消

**用户按 Ctrl+C**:
```
⚠️ **用户取消**

问题：你想要什么类型的游戏？

用户取消了提问。
```

---

### 3. 空回答

**required=true 且用户直接回车**:
```
⚠️ **未收到回答**

问题：你想要什么类型的游戏？

请重新提问或提供必填答案。
```
