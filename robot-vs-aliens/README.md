# 🤖 机器人大战外星人 🛸

一个使用 Python 和 Pygame 开发的经典射击游戏！

## 游戏特色

- 🎮 流畅的机器人控制
- 👾 多关卡外星人入侵
- 💥 炫酷的爆炸效果
- ⭐ 动态星空背景
- ❤️ 生命系统
- 🏆 分数统计

## 安装依赖

```bash
pip install pygame
```

## 运行游戏

```bash
python main.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → ↑ ↓ | 移动机器人 |
| 空格键 | 射击 |
| P 键 | 暂停游戏 |
| Q 键 | 退出游戏 |
| R 键 | 重新开始（游戏结束后） |

## 游戏规则

1. 控制机器人消灭所有外星人
2. 避免被外星人子弹击中
3. 防止外星人到达屏幕底部
4. 每消灭一波外星人，进入下一关
5. 关卡越高，外星人速度越快

## 游戏截图

游戏包含：
- 蓝色机器人（玩家）
- 绿色外星人（敌人）
- 黄色子弹（玩家射击）
- 紫色子弹（外星人射击）
- 爆炸特效
- 闪烁的星空背景

## 技术特点

- 使用 Pygame 精灵系统
- 碰撞检测
- 粒子爆炸效果
- 动态难度系统
- 面向对象设计
