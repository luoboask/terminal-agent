# 🤖 机器人战斗游戏

一个回合制策略战斗游戏，使用 Python 3 开发。

## 🎮 游戏特性

- **4 种机器人类型**：战士型、坦克型、刺客型、医疗型
- **回合制战斗系统**：策略性地选择攻击、防御、治疗
- **能量管理系统**：合理使用能量释放强力技能
- **随机敌人**：每次遭遇不同的敌人机器人
- **状态条显示**：直观显示 HP 和能量状态

## 🚀 快速开始

### 运行游戏

```bash
cd robot_battle_game
python3 robot_battle.py
```

### 游戏操作

1. **创建机器人**：输入名字并选择类型
2. **战斗行动**：
   - `1` - 普通攻击（恢复 15 能量）
   - `2` - 特殊攻击（消耗 30 能量，高伤害）
   - `3` - 防御（恢复 20 能量，减少伤害）
   - `4` - 治疗（消耗 25 能量，恢复 30 HP）
3. **击败敌人**：将敌人 HP 降为 0
4. **再玩一次**：选择是否开始新战斗

## 🤖 机器人类型

| 类型 | HP | 攻击 | 防御 | 特点 |
|------|-----|------|------|------|
| ⚔️ 战士型 | 120 | 25 | 15 | 均衡型，适合新手 |
| 🛡️ 坦克型 | 150 | 18 | 25 | 高防御，持久战 |
| ⚡ 刺客型 | 90 | 35 | 8 | 高攻击，高风险 |
| 💚 医疗型 | 110 | 20 | 15 | 可治疗，灵活 |

## 💡 游戏策略

1. **能量管理**：普通攻击和防御可以恢复能量
2. **时机把握**：低血量时使用治疗，能量充足时使用特殊攻击
3. **防御策略**：预测敌人攻击时使用防御
4. **类型克制**：根据敌人类型选择合适的机器人
