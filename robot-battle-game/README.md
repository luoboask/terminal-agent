# 🤖 机器人大战游戏

一个基于终端的回合制机器人大战游戏！

## 🎮 游戏特色

- **多种机器人类型**：突击型、坦克型、狙击型、平衡型
- **丰富的技能系统**：普通攻击、激光炮、导弹连射、能量护盾、修复程序、电磁脉冲
- **多种游戏模式**：人机对战、玩家对战、AI 对战
- **回合制战斗**：策略性的战斗体验
- **能量系统**：合理管理能量释放强力技能

## 🚀 快速开始

### 方法 1：直接运行
```bash
cd robot-battle-game
python3 main.py
```

### 方法 2：使用启动脚本
```bash
cd robot-battle-game
python3 run_game.py
```

## 📖 游戏说明

### 机器人类型

| 类型 | 特点 | 适合玩法 |
|------|------|----------|
| 突击型 | 高攻击，中等血量 | 适合进攻 |
| 坦克型 | 超高血量，高防御 | 适合防守 |
| 狙击型 | 超高攻击，低血量 | 玻璃大炮 |
| 平衡型 | 各项均衡 | 适合新手 |

### 技能说明

| 技能 | 伤害 | 命中率 | 冷却 | 效果 |
|------|------|--------|------|------|
| 普通攻击 | 15 | 95% | 0 | 基础攻击 |
| 激光炮 | 25 | 80% | 2 | 高伤害激光攻击 |
| 导弹连射 | 20 | 85% | 1 | 连续发射导弹 |
| 能量护盾 | 0 | 100% | 3 | 获得护盾保护 |
| 修复程序 | 治疗 | 100% | 3 | 恢复 25% 生命值 |
| 电磁脉冲 | 10 | 70% | 2 | 降低敌人命中率 |

### 操作指南

1. 启动游戏后选择游戏模式
2. 创建你的机器人（输入名字和选择类型）
3. 每回合选择行动：
   - 普通攻击（无冷却）
   - 使用技能（注意冷却时间）
   - 防御姿态（本回合防御加倍）
4. 击败对手赢得胜利！

## 🎯 胜利条件

将对手的生命值降至 0 即可获胜！

## 🎮 游戏模式

1. **人机对战**：玩家 vs 电脑 AI
2. **玩家对战**：玩家 1 vs 玩家 2（本地轮流操作）
3. **AI 对战**：电脑 vs 电脑（自动演示）

## 💡 游戏技巧

- 合理管理技能冷却时间
- 血量低时及时使用修复程序
- 能量护盾可以吸收大量伤害
- 电磁脉冲可以降低敌人命中率
- 根据敌人类型选择合适的战术
