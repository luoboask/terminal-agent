# 🐾 养成式宠物系统

一个简单有趣的命令行宠物养成游戏！

## ✨ 功能特性

- 🍖 **喂食系统** - 保持宠物饱食度
- 🎮 **互动玩耍** - 提升宠物心情
- 📚 **训练学习** - 增加经验值升级
- 😴 **休息恢复** - 恢复体力和健康
- 📊 **状态查看** - 实时查看宠物各项指标
- 💾 **自动存档** - 退出时自动保存进度
- 🎯 **成就系统** - 解锁各种成就

## 🚀 快速开始

```bash
# 运行游戏
python main.py
```

## 🎮 游戏指令

| 指令 | 功能 |
|------|------|
| 1 | 喂食 🍖 |
| 2 | 玩耍 🎮 |
| 3 | 训练 📚 |
| 4 | 休息 😴 |
| 5 | 查看状态 📊 |
| 6 | 保存游戏 💾 |
| 7 | 退出游戏 👋 |

## 📈 宠物属性

- **饱食度** (0-100): 过低会影响健康
- **心情值** (0-100): 影响互动效果
- **健康值** (0-100): 归零宠物会生病
- **体力值** (0-100): 进行活动需要消耗
- **经验值**: 积累到一定程度可升级
- **年龄**: 随时间增长，影响进化

## 🏆 进化阶段

1. 🥚 幼年期 (等级 1-5)
2. 🐾 成长期 (等级 6-10)
3. 🌟 成熟期 (等级 11-20)
4. 👑 完全体 (等级 21+)

## 💡 游戏提示

- 保持各项属性平衡发展
- 定期与宠物互动避免心情下降
- 注意宠物的体力，不要过度训练
- 宠物会随时间自然消耗属性，记得定期照顾！
