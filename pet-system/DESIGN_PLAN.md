# 🎨 PyQt6 macOS 风格宠物系统设计方案

## 📋 项目概述

将现有的终端宠物系统升级为现代化的 PyQt6 GUI 应用，采用 macOS 设计风格，提供流畅的动画效果和直观的用户体验。

---

## 🎨 macOS 设计规范

### 颜色方案
```python
# 主色调
PRIMARY_BLUE = "#007AFF"      # macOS 蓝
PRIMARY_GREEN = "#34C759"     # 成功绿
PRIMARY_RED = "#FF3B30"       # 警告红
PRIMARY_ORANGE = "#FF9500"    # 强调橙

# 背景色
BG_PRIMARY = "#FFFFFF"        # 主背景
BG_SECONDARY = "#F2F2F7"      # 次级背景
BG_TERTIARY = "#E5E5EA"       # 第三背景

# 文字色
TEXT_PRIMARY = "#000000"      # 主文字
TEXT_SECONDARY = "#3C3C43"    # 次级文字
TEXT_TERTIARY = "#8E8E93"     # 提示文字
```

### 圆角规范
- 大卡片：16px
- 中卡片：12px
- 小卡片：8px
- 按钮：10px
- 进度条：6px

### 阴影规范
```css
/* 轻阴影 */
box-shadow: 0 2px 8px rgba(0,0,0,0.08);

/* 中阴影 */
box-shadow: 0 4px 16px rgba(0,0,0,0.12);

/* 重阴影 */
box-shadow: 0 8px 32px rgba(0,0,0,0.16);
```

### 动画规范
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)
- 默认时长：250ms
- 强调动画：350ms
- 快速反馈：150ms

## 🔧 技术要点

### 1. 样式表 (QSS)
使用 Qt Style Sheets 实现 macOS 风格，类似 CSS。

### 2. 属性动画 (QPropertyAnimation)
实现平滑的数值变化动画。

### 3. 图形视图框架 (QGraphicsView)
用于宠物动画和特效渲染。

### 4. 信号槽机制
组件间解耦通信。

### 5. 多线程
耗时操作（如存档）不阻塞 UI。
