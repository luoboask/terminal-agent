# 机器人格斗游戏 (Robot Fighting Game)

一个基于终端的机器人格斗模拟游戏。

## 功能特性

- 🤖 多种机器人类型
- ⚔️ 实时战斗系统
- 📊 属性成长系统
- 🎮 简单的命令行界面

## 快速开始

```bash
python main.py
```

## 游戏说明

1. 选择你的机器人
2. 选择对手
3. 进行战斗
4. 赢取奖励并升级

## 项目结构

```
robot-fighting/
├── main.py          # 主程序入口
├── robot.py         # 机器人类定义
├── battle.py        # 战斗系统
├── config.py        # 游戏配置
└── README.md        # 说明文档
```
