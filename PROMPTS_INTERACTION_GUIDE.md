## 📖 使用方式

### 选择题模式

**调用方式**:
```bash
./start.sh "问用户：你想要什么类型的游戏？选项：回合制策略、实时动作、角色扮演"
```

**显示效果**:
```

  › 角色扮演
```

## 🔧 技术细节

### 工具参数

| 参数 | 类型 | 说明 |
|------|------|------|
| **question** | string | 要问的问题（必填） |
| **options** | string[] | 选项列表（可选，有则为选择题） |
| **required** | boolean | 是否必须回答（默认 true） |
| **initial** | number | 默认选中的选项索引（默认 0） |

---

### 返回数据

```typescript
{
  success: true,
  content: "✅ **用户回答**\n...",
  data: {
    question: "问题内容",
    answer: "用户回答",
    options: ["选项 1", "选项 2"],
    required: true
  }
}
```

## ⚠️ 注意事项

### 1. 交互会阻塞

**说明**: 使用 prompts 会暂停程序等待用户输入

**影响**: 
- ✅ 真正的交互
- ⚠️ 会阻塞当前会话

---

### 2. 用户取消处理

**用户按 Ctrl+C**:
```
⚠️ **用户取消**

问题：你想要什么类型的游戏？

用户取消了提问。
```

---

### 3. 空回答处理

**required=true 时**:
```
⚠️ **未收到回答**

问题：你想要什么类型的游戏？

请重新提问或提供必填答案。
```
