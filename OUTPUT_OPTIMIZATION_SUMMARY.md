## 🔧 优化内容

### 1. 移除工具调用统计

```typescript
// ❌ 删除
console.log(`📊 工具调用统计：${buffer.length} 次工具调用完成`);
```

**原因**: 每次响应都显示，太冗余

---

### 2. 简化参数显示

```typescript
// ❌ 优化前
console.log(`   参数：${JSON.stringify(chunk.toolInput, null, 2).slice(0, 200)}`);

// ✅ 优化后
console.log(`\n🔧 使用 ${chunk.toolName || '工具'}...`);
```

**原因**: 参数详情不需要每次都显示

---

### 3. 简化结果显示

```typescript
// ❌ 优化前
console.log(`${chunk.content.slice(0, 300)}...`);

// ✅ 优化后
console.log(`✅ ${chunk.toolName || '工具'} 执行完成\n`);
```

**原因**: 结果由 AI 总结显示，不需要重复
